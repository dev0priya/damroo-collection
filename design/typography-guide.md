# Typography Guide

## Fonts
- **Headlines**: 'Noto Serif', serif (Editorial authority, Vogue-esque).
- **Body & UI**: 'Manrope', sans-serif (Modern, functional, geometric).

## Scales
- **Headline XL**: 48px / 3rem (Line Height: 1.2, Weight: 700) - Hero Sections.
- **Headline LG**: 32px / 2rem (Line Height: 1.3, Weight: 600) - Section Headings.
- **Headline MD**: 24px / 1.5rem (Line Height: 1.4, Weight: 600) - Product/Category Cards.
- **Body LG**: 18px / 1.125rem (Line Height: 1.6, Weight: 400) - Large descriptive text.
- **Body MD**: 16px / 1rem (Line Height: 1.5, Weight: 400) - Default body copy.
- **Label LG**: 14px / 0.875rem (Line Height: 1.2, Weight: 600) - Upper-case tags, navigation.
- **Label MD**: 12px / 0.75rem (Line Height: 1.2, Weight: 500) - Metadata, captions.

## Hierarchy
- Use Uppercase with 0.05em letter-spacing for **Labels**.
- Use tight tracking for **Large Headlines**.
