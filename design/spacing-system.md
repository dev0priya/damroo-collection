# Spacing System

The spacing system is governed by an 8px rhythmic scale.

## Scale
- **XS**: 0.25rem (4px)
- **SM**: 0.5rem (8px)
- **MD**: 1rem (16px) - Standard gutter.
- **LG**: 1.5rem (24px) - Component spacing.
- **XL**: 2.5rem (40px) - Section spacing.

## Layout
- **Container Max**: 1280px.
- **Gutter**: 1.5rem (24px) on desktop.
- **Mobile Gutter**: 1rem (16px).

## Radius
- **Base (Level 1)**: 4px (0.25rem) - Default for buttons, cards, inputs.
- **Sharp**: 0px - Hero banners and promotional sections.
