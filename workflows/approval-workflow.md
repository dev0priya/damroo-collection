# Approval Workflow

Process for obtaining user feedback and approval.

## Protocol
1. **Implementation Plan**: For any non-trivial change, create `implementation_plan.md`.
2. **User Review**: Submit the plan and wait for the user to review.
3. **Feedback Integration**: Update the plan based on user comments.
4. **Final Approval**: Proceed with execution ONLY after explicit approval.
5. **Walkthrough**: After completion, provide a `walkthrough.md` with screenshots/videos if applicable.
